System Log Options

Manage System Log

 

Most events in TeamCal can be logged so history information is available if needed. With this dialog here the administrator can switch on or off nine different event types for logging:

 

•Log Absence Type events
Switching this on will log all absence type activities (add, update, delete)

 

•Log Announcement events
Switching this on will log all announcement activities, e.g. confirmations by users
 

•Log Configuration changes
Switching this on will log all configuration changes by the administrator accounts
 

•Log Database Maintenance events
Switching this on will log all absence type activities (add, update, delete)

 

•Log Daynote events
Switching this on will log all general day note activities (add, update, delete)

 

•Log Group events
Switching this on will log all group activities (add, update, delete)

 

•Log Holiday events
Switching this on will log all holiday activities (add, update, delete)

 

•Log Login events
Switching this on will log all login/logout activities

 

•Log Loglevel changes
Switching this on will log all changes to the log level settings

 

•Log Month events
Switching this on will log all month template activities

 

•Log User events
Switching this on will log all user related activities (add, update, delete, personal day note, templates)
 

•Log Region events
Switching this on will log all region changes
 

•Log Registration events
Switching this on will log all registration events
