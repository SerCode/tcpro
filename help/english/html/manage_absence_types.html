Absence Types

Manage Absences

 

The Manage Absence Types page can be used to configure each absence type to your requirements. You can change the absence symbol, name, text color and background color and other settings. You can only update or delete one absence type at a time.

 

With the select box at the top you can choose the absence type you want to edit. Right next to it you can add a name for a new absence type and click [Create] to create it. It will then be loaded into the editor.

 

Explanations for each setting are provided directly on the page.

Buttons

•[Apply]
Applies the changes in the editor to the currently selected absence type.
 

•[Create]
Creates a new absence type with the name provided in the text box.

 

•[Delete]
Deletes the currently selected absence type. A warning message will be displayed.

     

•[Help]
Shows this help page
 

•[Upload]
Opens the Upload dialog so can upload a custom icon for your absence type.
