View Profile Dialog

 

The view user profile dialog shows all relevant user information and offers the option to send a mail to this user directly from your browser.

 

The following buttons are available:

Buttons

•[Send e-Mail]
Send the text in the Message field as an e-Mail to this user using the Subject text as subject.

     

•[Help]
Will display this section of this manual.

     

•[Close]
Will close the dialog.
