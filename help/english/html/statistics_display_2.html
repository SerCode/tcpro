Remainder Current Year

This statistics page offers a visual representation of the remainders of each absence type for each user in the current year. Only absence types with a maximum allowance set are included (Note: If you don't see an absence type here its allowance is probably set to 0.).

 

TeamCal Pro will loop through each group and in each group through each user. A diagram is created for each user. You can narrow the selection by choosing a specific group or user only.

 

 

Users might have a remainder from the previous year. That value is also shown in the table. The diagram shows the total remainder from last year and the current year. You cannot select the year since the allowance is always computed against the current year plus the remainder of the last year.
